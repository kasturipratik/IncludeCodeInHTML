Connecting Database to your Java Application (JDBC)

 

Steps for connecting your Java application the database

  • Download and install Intellij Ultimate IDE
    Show Image
  • Download and install JDBC MySql Connector for your Operating System or any connector applicable to you.
    Show Image
  • Download and install Mysql workBench or MySql Installer
    Show Image
  • Open Intellij IDE and create new project.
    • Click Java Enterprise
    • Select the tomcat server you downloaded
    • Navigate to the Additional Libraries and Frameworks and select web application
    • Click next and give your project a name and click finish
